# 💊 Medical Store Management System

## 🚀 Features
- Admin & Operator Login
- Medicine Inventory Management
- Expiry-based filtering
- Sales management system
- Date-wise sales chart (Chart.js)
- LocalStorage + MongoDB integration

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB
- Chart.js

## 📊 Key Highlights
- Batch-wise expiry handling
- Real-time stock validation
- Dynamic sales chart

## ▶️ How to Run
1. npm install
2. node server.js
3. Open http://localhost:3000
